# CNN-PyTorch
This repository contained Google Colab notebooks with PyTorch implementations of convolutional neural networks(CNNs) and its variants.
- The first notebook has a simple CNN model implemented for image classification on the MNIST dataset.
- The second notebook describes the UNet model and how it is used for image segmentaTION.
- The third showcases the YOLO architecture, and describes how it can be used for object detection.
